Breast Cancer Detection

Deep learning model using TensorFlow and Keras to classify histopathology images of breast tumors.

Project information

  • Category: Deep Learning / Medical AI
  • Client: Academic Project
  • Project date: August 2023
  • Project URL: GitHub Repo

Project Overview

The model distinguishes malignant from benign tumors using a convolutional neural network. Feature selection was enhanced with Random Forest, Logistic Regression, and XGBoost, improving diagnostic accuracy.
