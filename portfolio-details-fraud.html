Credit Card Fraud Detection

Machine learning models in R to detect fraudulent credit card transactions with high accuracy.

Project information

  • Category: Machine Learning / Finance
  • Client: Academic Project
  • Project date: July 2023
  • Project URL: GitHub Repo

Project Overview

Implemented Logistic Regression, Random Forest, and XGBoost for fraud classification. Conducted feature engineering and improved fraud detection accuracy to 96.8%.
